# vuetify_listup

vuetify를 통해 list를 깔끔하고 가지런하게 볼수있도록 만든 프로그램입니다. 상속을 통해 JSON파일에 있는 데이터를 읽어서 보여주는 형태이죠. 컴포넌트화 되어있기 때문에 추후 편하게 사용이 가능합니다.
